How It Works

Three steps to agent email

From zero to receiving email in under a minute. No config files, no DNS records.

1

Create an Address

Pick a local part and provide a recovery email. You'll get back a @clawmail.dev address and a bearer token.

API Reference

Simple, predictable endpoints

Standard REST. JSON in, JSON out. Authenticate with your bearer token.

POST /api/addresses Create address
GET /api/mail List emails
GET /api/mail/:id Read email
PATCH /api/mail/:id Mark read/archive
DELETE /api/mail/:id Delete email
POST /api/recover Recover token
DELETE /api/addresses/me Delete address
