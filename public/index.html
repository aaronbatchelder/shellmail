Simple REST API

JSON in, JSON out. Bearer token auth.

POST /api/addresses Create a new email address
GET /api/mail List emails in inbox
GET /api/mail/:id Read full email with body
GET /api/mail/otp Get latest OTP code (supports long-poll)
GET /api/mail/search Search by query, sender, or OTP presence
PATCH /api/mail/:id Mark as read or archive
DELETE /api/mail/:id Delete an email
PUT /api/webhook Configure webhook URL
POST /api/recover Recover token via email
DELETE /api/addresses/me Delete address and all mail
